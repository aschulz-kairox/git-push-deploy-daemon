## ğŸ”§ Integration with GPD

Configure GPD to use GPDD instead of PM2:

```json
{
  "services": {
    "my-api": {
      "processManager": "gpdd",
      "gpddWorkers": 4,
      "gpddEntryPoint": "dist/index.js"
    }
  }
}
```

Then use GPD's daemon command:

```bash
gpd daemon my-api start    # Start
gpd daemon my-api reload   # Zero-downtime reload
gpd daemon my-api status   # Show status
gpd daemon my-api stop     # Stop
gpd daemon my-api logs     # Show logs
```

## ğŸ“ Your App Requirements

For zero-downtime reload to work, your app needs to signal when it's ready:

```javascript
// dist/index.js
const http = require('http');

const server = http.createServer((req, res) => {
  res.end('Hello World');
});

server.listen(process.env.PORT || 3000, () => {
  // Signal to master that this worker is ready
  if (process.send) {
    process.send('ready');
  }
  console.log(`Worker ${process.pid} listening`);
});

// Handle graceful shutdown
process.on('message', (msg) => {
  if (msg === 'shutdown') {
    console.log(`Worker ${process.pid} shutting down...`);
    server.close(() => {
      process.exit(0);
    });
  }
});
```
